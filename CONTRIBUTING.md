# Contributing to ProtoBot

Thank you for your interest in ProtoBot.

## License and use

This repository is distributed under **CRCL-ProtoBot-v1** (see [LICENSE](LICENSE) and [LICENSE.md](LICENSE.md)). Contributions must respect those terms: commercial use is limited to authorized holders; everyone else may contribute for **non-commercial** use only, with full attribution.

## Attribution in source files

Every new source file should include the header defined in [docs/templates/license-header.txt](docs/templates/license-header.txt) (or an equivalent comment style for the language).

## Suggested workflow

1. Open an issue describing the change.
2. Create a branch from `main`.
3. Keep commits focused with clear messages.
4. Open a pull request and link the issue.
